Overview
This is a simple Windows Forms quiz application built in VB.NET. The program allows users to create, edit, and take multiple-choice quizzes through a graphical interface.
Features
Add new quiz questions
Edit existing questions
Take a multiple-choice quiz
Immediate feedback on answers
Easy-to-use GUI
How It Works
Questions are stored in memory while the program is running. Each question contains the question text, four answer choices, and the correct answer.
Running the Program
Open the solution in Visual Studio
Build and run the project
No additional setup is required.
